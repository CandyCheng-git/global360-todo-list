# Project Instructions

Build a simple TODO list application for a technical assessment.

## Stack
- Frontend: latest stable Angular using standalone components
- Backend: latest stable .NET Web API
- Data: backend in-memory storage only
- Testing: include meaningful unit/API tests
- Packaging: Docker and docker-compose

## Requirements
- User can view TODO list
- User can add TODO item
- User can delete TODO item
- Backend manages TODO data in memory
- Keep the app simple and clean
- Do not add authentication, database, user accounts, or unnecessary features

## Code Quality
- Use clear naming
- Keep components/services small
- Add validation
- Add error handling
- Add README run instructions
- Ensure all tests pass before final submission
